## The short answer

Procurement does not need to become a GD&T trainer. It does need to know which callouts change price, inspection method, and supplier capability.

## Why GD&T matters in sourcing

GD&T defines allowable variation relative to datums and function. That is usually better than loose coordinate dimensioning, but only when the callouts are used correctly. Bad GD&T is worse than no GD&T because it looks precise while creating inspection fights.

A buyer should know whether the print relies on position, profile, flatness, parallelism, perpendicularity, runout, circularity, cylindricity, or concentricity. Those choices change both process and metrology.

## The 10 callouts you will see most

Position controls feature location. Flatness controls a plane without a datum. Parallelism and perpendicularity tie features to a datum. Profile controls complex surfaces. Runout applies to rotating features. Circularity and cylindricity control round features directly. Concentricity is rarer and often overused. True design intent usually reads better with position or runout.

The question is not whether the symbol looks familiar. It is whether the supplier can machine and inspect it with confidence.

## Where cost spikes

Profile on complex surfaces, tight positional tolerance at MMC or LMC, and broad datum structures can push the part into CMM-heavy inspection and more process control. That may be justified. It may also be overkill.

A buyer should challenge callouts that duplicate function already covered elsewhere. Stacked precision language is not sophistication. It is often just noise.

## How to read the print like a buyer

Ask which callouts define fit, motion, sealing, or assembly alignment. Ask which datums are actual functional contacts and which were chosen for drafting convenience. If the answer is weak, the drawing probably needs a DFM review before quoting.

That step is faster than fighting NCRs later.

## Comparison table where relevant

| Callout | What it controls | Buyer watchout |
| -- | -- | -- |
| Position | Feature location to datums | Often inspection-heavy |
| Profile | Surface envelope | Can be expensive on complex forms |
| Flatness | Single plane | No datum needed |
| Runout | Rotating feature variation | Use on true functional rotation |
| Concentricity | Median point relation | Often overcalled |


## How to specify this in your RFQ

On the RFQ, identify the few GD&T callouts tied to function and ask the supplier to flag any that drive special inspection. If a CMM report is required, state that early. Do not bury it in the PO terms after the quote is accepted.

A clean RFQ does not just list requirements. It separates must-haves from preferences so the supplier can optimize where it is safe.
